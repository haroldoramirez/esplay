1 Introdução

Este documento tem por objetivo servir como um manual de instruções, de forma a dar fundamentos para o usuário operar o sistema web Esplay. Além disso, demonstrar as características do sistema, bem como suas telas e funcionalidades.

2 Sistema Esplay

O objetivo deste produto é ajudar as pessoas gerenciar seu compromissos de maneira prática. Trata-se de um sistema simples e fácil de usar. O sistema conta com a opção de gerenciar usuários, agenda compromissos, criar categorias de compromissos, criar tipos de compromissos, gerenciar contatos e mostrar os logs que acontecem no sistema..

Todo o sistema Esplay tem a intenção de ser facilmente usado através da interface WEB.
